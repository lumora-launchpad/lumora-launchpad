## Launch playbook (two weeks)

A concrete order of operations so the public launch does not land on an empty
site. Lumora is on testnet, so frame everything as "early access", and never
imply it is audited or on mainnet.

**Week 1, set the stage**

1. Create the X account, set the bio, banner, and avatar, and pin the intro
   thread. Set the location to Base and the link to lumora.men.
2. Seed the site (see the seed plan below) so there are 6 to 10 real tokens and
   2 campaigns before anyone arrives. The ticker, King of the hill, and live
   activity feed should all be populated.
3. Make a Telegram group and put the link in the X bio.
4. Soft launch to people you know. Ask 5 to 10 friends to each make one trade,
   so the live activity feed and holder counts are non-zero.

**Week 2, go public**

5. Post the intro thread publicly and ask your network to repost the first post.
6. Recruit 3 to 5 creators with the outreach DM below. Help each one launch a
   campaign and post it.
7. Be present in Base communities. Share, do not spam. Answer questions.
8. Post every launch, funded campaign, and graduation with its link, so the OG
   card renders. One post each, same day.
9. Keep a daily rhythm for two weeks. Distribution is repetition.
